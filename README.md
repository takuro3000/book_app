## 概要

TechBook Finder は「自分に合った難易度のプログラミング本との出会い」をサポートするために開発したアプリです。

技術書には幅広い難易度があり、「思っていたより難しくて挫折した」「簡単すぎて得るものがなかった」といった経験をしたことがある人も多いでしょう。TechBook Finder では、ユーザーによる難易度レビューをもとに、自分のスキルレベルに合った技術書を見つけることができます。

## ペルソナ

### ペルソナ 1: 山本 太郎

- **年齢・性別**
  22 歳、男性
- **職業・バックグラウンド**
  大学で文系を専攻中。最近、プログラミングに興味を持ち、将来のキャリアチェンジや副業としての可能性を探るために学習を始めようとしている。プログラミング未経験のため、どの入門書から始めればよいのか情報が多すぎて戸惑いを感じている。
- **モチベーション・目標**
  - 初心者にとって無理のない、基礎から学べる最適な入門書を見つけたい。
  - 効率的にプログラミングの基本を身につけ、将来的なスキルアップやキャリアの幅を広げたい。
- **悩み・フラストレーション**
  - インターネット上に膨大な情報があり、どの書籍が自分のレベルに合っているのか判断が難しい。
  - 書籍ごとの難易度や学習効果が具体的に分からず、購入前に不安を感じる。
- **利用シーン**
  - 書籍選びのため、実際に同じ立場の初心者ユーザーが投稿した具体的なレビューや体験談を参考にしたい。
  - 書籍ごとの難易度や学習進度の目安が一目で比較できる情報が欲しい。

### ペルソナ 2: 佐藤 美香

- **年齢・性別**
  34 歳、女性
- **職業・バックグラウンド**
  IT 企業で業務に携わる中級レベルのエンジニア。入社当初は独学でプログラミングを学び、その後実務を通じてスキルを磨いてきた。現在は、より深い知識や考え方を得るために、プログラミングの名著と呼ばれる書籍に手を出したいと考えているが、どの本が自分の現状のスキルに適しているのか迷っている。
- **モチベーション・目標**
  - 長年語り継がれるプログラミングの古典的名著を読み込み、コードの品質や設計思想などをさらに深く理解したい。
  - 自分に合った難易度の本を選ぶことで、無理なく知識を吸収し、業務に役立てたい。
- **悩み・フラストレーション**
  - 評判の高い名著が必ずしも自分にとって適切なレベルであるとは限らず、レビュー情報だけでは判断しづらい。
  - 自分と同じ経験や背景を持つユーザーの具体的な感想が少なく、どこに重点を置くべきか迷ってしまう。
- **利用シーン**
  - 書籍ごとに「初心者向け」「中級者向け」「上級者向け」といった難易度評価や、実際の学習効果についての詳細なレビューをチェックする。
  - 同じく中級レベルのエンジニアたちの意見を参考にし、自分に合った一冊を絞り込みたい。

### ペルソナ 3: 高橋 修

- **年齢・性別**
  38 歳、男性
- **職業・バックグラウンド**
  大手企業のシニアエンジニア。数年の実務経験と複数のプロジェクト成功実績を持つが、常に新しい技術や手法を求め、自己研鑽に余念がない。最新の技術書、特にオライリーなどの新刊に強い関心があるが、内容があまりにも初心者向けであれば時間を割きたくないと考えている。
- **モチベーション・目標**
  - 自身の経験をさらに深化させるため、チャレンジングな内容や新たな視点を提供してくれる書籍を求めている。
  - 新刊の中から、自分のスキルレベルを一段と高めるために適した、上級者向けの本を厳選して読みたい。
- **悩み・フラストレーション**
  - 書籍の紹介文や表面的なレビューだけでは、本当に高度な内容が含まれているのか判断しにくい。
  - 初心者向けとされる内容と上級者向けの微妙な境界線が明確でなく、情報の取捨選択に時間がかかる。
- **利用シーン**
  - 最新の技術書やオライリーの新刊の詳細なレビュー、特に具体的な難易度や実務への応用例など、深堀りした情報を求める。
  - 自身と同じような上級エンジニアの意見や評価をもとに、効率的に読み進めるべき書籍を選定したい。
